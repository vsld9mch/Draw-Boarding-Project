# Draw Boarding Project #

A versatile drawing application developed to create geometric shapes and lines with precision.

# Key Features #
* **Dual Interface Modes:** Features both **standard (non-extended)** and **enhanced (extended)** drawing boards to suit different user needs.
* **Geometric Tools:** Specialized for rendering geometric primitives and linear elements.

# Tech Stack #
* **Language:** C#
* **IDE:** Visual Studio 2022
* **Developed:** January 2024

# About the Project #
This project acts as a functional drawing board, allowing users to experiment with shapes in two different workspace configurations. It was built as a practical implementation of graphics handling in C#.
