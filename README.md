# Draw Boarding Project

## Project Overview
Developed in **January 2024**, this project is a versatile drawing application designed for creating geometric shapes and lines with precision. It serves as a functional digital drawing board, allowing users to experiment with various geometric primitives within a graphical environment.

## Key Features
* **Dual Interface Modes**: The application offers two distinct workspace configurations:
    * **Standard Mode**: A basic drawing board for essential shapes.
    * **Enhanced Mode**: An extended version with expanded functionality for more complex work.
* **Geometric Focus**: Specialized tools for rendering and manipulating geometric primitives and linear elements.
* **Interactive Canvas**: Provides a workspace where users can visualize and test graphics handling logic in real-time.

## Technical Stack
* **Language**: C#
* **Platform**: .NET (Windows Desktop)
* **IDE**: Visual Studio 2022
* **Project Type**: Desktop Application (Graphics-focused)
